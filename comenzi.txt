GENERALE:
1. lista dosare in locatia curenta: DIR
2. goto dosar: CD /PATH/TO/FOLDER
3. creare dosar: MKDIR nume_dosar
4. delete fisier: rm -f .dosar/nume_fisier.extensie (sau: de -f .dosar/nume_fisier.extensie)
5. autoremove: sudo apt autoremove
6. verificare update: sudo apt update
7. instalare update: sudo apt upgrade

13. Docker: (https://docs.docker.com/engine/install/ubuntu/)
    1. Sudo apt-get install \
    ca-certificates \
    curl \
    gnupg \
    lsb-release
    2. curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
    3. echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
    4. sudo apt-get update
    5. sudo apt-get install docker-ce docker-ce-cli containerd.iosudo docker version
    6. verificare versiune: sudo docker version
